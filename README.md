## Opis

Jednostavna chrome extenzija koja šalje obavijesti za novosti s mev.hr


## Instalacija

1.Otvori stranicu za extenzije u Chrome-u(chrome://extensions/)
2.Na tu stranicu povucite jsRssReader.crx i instalirajte extenziju
3.Desni klik na ikonu extenziju odaberite options i na toj stranici odaberite vrstu obavijesti koju želte primati
